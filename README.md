# Fall Risk Environmental Audit Tool

This project is a web-based environmental audit tool designed to assess fall risks within home environments. It focuses on identifying external risk factors that contribute to falls, especially among elderly individuals.

## Features
- 30-item hazard checklist (flooring, lighting, bathroom, furniture, accessibility)
- Simple scoring system to calculate home risk level
- Risk classification: Low, Moderate, High
- Basic safety recommendations based on results
- Interactive web-based prototype

## Objective
To provide a holistic approach to fall risk assessment by analyzing environmental hazards in residential settings.

## Use Case
- Home safety evaluation
- Elderly care assessment
- Preventive healthcare research

## Tech Stack
- HTML
- CSS
- JavaScript

## Output
- Risk score (0–30)
- Risk category
- Suggested improvements

## Status
Prototype developed and pilot-tested on sample households.
